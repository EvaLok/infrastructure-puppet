# infrastructure-puppet

Puppet configuration for jQuery Infrastructure servers.

## Contributing

### Linting

See also <http://puppet-lint.com/checks/> and <https://github.com/rodjek/puppet-lint/>.

Install:
```
gem install puppet-lint -v '~> 2'
```

Run linter:
```
puppet-lint .
````

Autofix:
```
puppet-lint --fix .
```
